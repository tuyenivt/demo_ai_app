# Telehealth Virtual Care Assistant

This is a **sample Markdown file** for testing Qdrant upsert operations.

## About the Virtual Care Assistant
The Telehealth Virtual Care Assistant helps patients:
- Schedule appointments
- Conduct symptom triage
- Receive medication reminders
- Chat with an AI healthcare agent

## Benefits
1. Improves access to care
2. Reduces administrative burden
3. Enhances patient engagement

## Sample FAQ
**Q:** What conditions can the assistant help with?  
**A:** Non-emergency concerns such as cold/flu symptoms, medication questions, and monitoring chronic conditions.
